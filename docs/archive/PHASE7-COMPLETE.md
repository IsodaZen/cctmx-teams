## 🚀 本番環境での使用準備

**ステータス**: ✅ **本番環境で使用可能**

### 使用開始手順

1. **プラグインのインストール**
   ```bash
   cp -r /Users/z-isoda/private/cctmx-teams ~/.claude/plugins/
   ```

2. **tmuxセッションの作成**
   ```bash
   tmux new-session -s claude-dev
   ```

3. **Claude Code の起動**
   ```bash
   claude
   ```

4. **セットアップの実行**
   ```
   /cctmx-teams:setup
   ```

5. **使用開始**
   - `/tmux-worker`: ワーカーペイン作成
   - `/tmux-send`: タスク送信
   - `/tmux-review`: レビュー実行
   - `/tmux-check`: エラーチェック
